# /mykit.help

Show available My Kit commands.

## Usage

```
/mykit.help
```

## Commands

| Command | Description |
|---------|-------------|
| /mykit.init | Initialize My Kit |
| /mykit.setup | Configure preferences |
| /mykit.start | Start a workflow |
| /mykit.status | Show current status |
| /mykit.backlog | Manage backlog |
| /mykit.specify | Create specification |
| /mykit.plan | Generate plan |
| /mykit.tasks | Generate tasks |
| /mykit.implement | Execute tasks |
| /mykit.validate | Run validation |
| /mykit.commit | Create commit |
| /mykit.pr | Create pull request |
| /mykit.release | Create release |
| /mykit.resume | Resume workflow |
| /mykit.reset | Reset state |
| /mykit.upgrade | Upgrade My Kit |
| /mykit.help | Show this help |
