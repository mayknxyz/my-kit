# Canonical Labels

All commands and skills must use only these labels. Never create labels outside this list.

## GitHub Defaults

| Label | Description | Color |
|-------|-------------|-------|
| `bug` | Something isn't working | `d73a4a` |
| `documentation` | Improvements or additions to documentation | `0075ca` |
| `duplicate` | This issue or pull request already exists | `cfd3d7` |
| `enhancement` | New feature or request | `a2eeef` |
| `good first issue` | Good for newcomers | `7057ff` |
| `help wanted` | Extra attention is needed | `008672` |
| `invalid` | This doesn't seem right | `e4e669` |
| `question` | Further information is requested | `d876e3` |
| `wontfix` | This will not be worked on | `ffffff` |

## Custom Labels

| Label | Description | Color |
|-------|-------------|-------|
| `chore` | Deps, CI, config | `fef2c0` |
| `refactor` | Restructuring without behavior change | `d4c5f9` |
| `performance` | Perf improvements | `f9d0c4` |
| `security` | Security fixes | `b60205` |
| `a11y` | Accessibility fixes | `0e8a16` |
| `breaking` | Signals major version bump | `e11d48` |
| `p1-critical` | Drop everything — fires, outages | `b60205` |
| `p2-high` | Do next — important but not on fire | `d93f0b` |
| `p3-medium` | Planned — scheduled for upcoming work | `fbca04` |
| `p4-low` | Backlog — nice to have, no rush | `0e8a16` |

## Auto-Detection Keywords

Use these mappings when auto-detecting labels from content:

| Label | Keywords |
|-------|----------|
| `bug` | error, crash, broken, fix, fails, doesn't work, regression |
| `enhancement` | add, implement, support, new, feature, proposal |
| `documentation` | docs, readme, guide, typo in docs |
| `chore` | deps, dependency, CI, config, tooling, build |
| `refactor` | refactor, cleanup, restructure, reorganize |
| `performance` | slow, performance, optimize, memory, latency, speed |
| `security` | vulnerability, CVE, auth, XSS, injection, security |
| `a11y` | accessibility, WCAG, screen reader, aria, keyboard navigation |
| `breaking` | breaking change, major version, incompatible |
